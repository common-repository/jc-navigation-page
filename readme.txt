JC Navigation Page to switch quickly from one page to another

== Description ==

JC is a small plugin. In the edit page, it displays a list of links to pages related to the page being edited: parent page, adjacent pages, child pages.
This plugin is very useful on sites with many pages and saves valuable time.

[more information on JC Navigation Page plugin](http://www.netcod.es/)

**be more effective**

We have also released an advanced plugin based on JC Navigation Page named [Pronto](http://cut.lu/pronto) which adds some important features :

* also works on post and media pages.
* search box 
* preview box of the element.
* advanced actions : view, edit and insert into content.

For more information, [visit our plugin website](http://www.netcod.es/products/) or [Pronto page](http://cut.lu/pronto)

== Installation ==

1. Upload the folder `jc-nav-page` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
